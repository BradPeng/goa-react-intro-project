# Scope

Predict what you get on your console when you execute the following code.

1.

```js
var a = 1;
console.log(this.a);
```

2.

```js
let b = 2;
console.log(this.b);
```

3.

```js
const c = 3;
this.c;
console.log(this.c);
```


4.

```js
var a1 = a2 = 1;
console.log(this.a1);
console.log(this.a2);
```

5.

```js
let b1 = b2 = 2;
console.log(this.b1);
console.log(this.b2);
```

6.

```js
const c1 = c2 = 3;
console.log(this.c1);
console.log(this.c2);
```
