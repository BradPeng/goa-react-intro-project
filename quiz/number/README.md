# Number

Predict what you get on your console when you execute the following code.

1.

```js
const sum = (a, b = 2) => a + b;
console.log(sum(1));
console.log(sum(1, 0));
console.log(sum(1, null));
console.log(sum(1, undefined));
```

2.

```js
console.log(5 & 1);
console.log(5 & 2);
console.log(5 & 3);
```

3.

```js
console.log(5 | 1);
console.log(5 | 2);
console.log(5 | 3);
```

4.

```js
console.log(5 ^ 1);
console.log(5 ^ 2);
console.log(5 ^ 3);
```
